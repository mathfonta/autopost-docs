## Story
**As a** profissional que grava vídeos de obra enquanto narra o que está fazendo,
**I want** que o AutoPost entenda o que eu falei no vídeo,
**so that** a legenda gerada reflita o contexto real do serviço — sem eu precisar digitar nada.

## Contexto / Motivação
O copywriter hoje recebe apenas descrição visual dos frames do vídeo (Claude Haiku analisa
imagens estáticas extraídas com FFmpeg). Se o profissional narra o serviço enquanto filma
— o que é muito comum — esse contexto é completamente perdido.

Exemplo real (vídeo da porta de correr, 45s):
- O que o analista visual vê: "porta de madeira escura sendo instalada em banheiro"
- O que o profissional provavelmente narrou: "a cliente chegou com o trilho mas a porta não
  servia, adaptamos uma porta maciça de cozinha, cortamos na medida, instalamos a fechadura..."

Esse segundo nível de contexto é ouro para o copywriter — é exatamente o tipo de detalhe
que transforma uma legenda genérica numa legenda de alta conversão.

Solução: extrair o áudio do vídeo com FFmpeg (já disponível no Dockerfile) e transcrever
com OpenAI Whisper API. A transcrição é injetada no copywriter como contexto adicional.

## Acceptance Criteria

### Backend — Transcrição
1. **AC1 — Extração de áudio:** Em `analyze_video_with_ai()`, antes de processar os frames,
   o backend extrai o áudio do vídeo com FFmpeg: `ffmpeg -i input.mp4 -vn -acodec pcm_s16le -ar 16000 -ac 1 output.wav`
2. **AC2 — Transcrição Whisper:** O arquivo de áudio é enviado à OpenAI Whisper API
   (`model="whisper-1"`, `language="pt"`) e a transcrição retornada é armazenada em
   `analysis_result["audio_transcript"]`.
3. **AC3 — Fallback gracioso:** Se a extração ou transcrição falhar (FFmpeg error, API error,
   timeout, áudio silencioso), o pipeline continua normalmente sem transcrição — nunca bloqueia.
4. **AC4 — Filtro de qualidade:** Se a transcrição tiver menos de 20 palavras (vídeo sem
   narração útil, apenas ruído), o campo `audio_transcript` é definido como `None` e não
   é injetado no copywriter.
5. **AC5 — Injeção no copywriter:** Em `generate_copy_with_ai()`, se `analysis_result` contiver
   `audio_transcript` não-nulo, o conteúdo é injetado no user message com o label:
   `"TRANSCRIÇÃO DO ÁUDIO (o profissional narrou durante o vídeo):\n{transcript}"`

### Backend — Configuração
6. **AC6 — Variável de ambiente:** A chave OpenAI é lida de `OPENAI_API_KEY` via Settings.
   Se a variável não estiver definida, a transcrição é silenciosamente ignorada (fallback).
7. **AC7 — Custo controlado:** A transcrição só é executada para `content_type in ("reels", "story")`.
   Fotos não são afetadas.

### Qualidade
8. **AC8 — Sem regressão:** O fluxo de fotos (feed_photo, carousel, before_after) não é afetado.
9. **AC9 — Testes:** `test_audio_transcription.py` com pelo menos 3 casos:
   vídeo com narração (transcript retornado), vídeo silencioso (filtrado), Whisper API indisponível (fallback).

## Scope

**IN:**
- `backend/app/agents/analyst.py` — `analyze_video_with_ai()`: extrai áudio + chama Whisper
- `backend/app/agents/copywriter.py` — `generate_copy_with_ai()`: injeta `audio_transcript` se presente
- `backend/app/config.py` — adicionar `OPENAI_API_KEY: str = ""` (opcional)
- `backend/tests/test_audio_transcription.py` — 3+ casos de teste
- Railway: adicionar `OPENAI_API_KEY` nas variáveis de ambiente do serviço `worker`

**OUT:**
- Transcrição de áudio em fotos
- Edição/exibição da transcrição no frontend
- Armazenamento permanente da transcrição (apenas em `analysis_result` JSON)
- Tradução da transcrição para outro idioma

## Business Value
Elimina a lacuna entre "o que o profissional sabe sobre o serviço" e "o que o AutoPost consegue
gerar". Para Reels — o formato de maior alcance orgânico — a melhoria na qualidade da copy
pode ser a diferença entre 500 e 5000 visualizações.

Custo operacional: ~$0.005 por vídeo de 45s (Whisper: $0.006/min). Desprezível.

## Risks
| Risco | Probabilidade | Mitigação |
|-------|--------------|-----------|
| Vídeo sem narração (só ruído/música) | Alta | Filtro de 20 palavras mínimas (AC4) |
| OPENAI_API_KEY não configurada em produção | Média | Fallback silencioso (AC6) |
| Whisper latência alta (> 10s) | Baixa | Timeout 30s + fallback |
| Custo inesperado com vídeos longos | Baixa | Transcrição só para reels/story; vídeos já limitados a 45s idealmente |
