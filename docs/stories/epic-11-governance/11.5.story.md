# Story 11.5 — Smoke Test de Pipeline

**Epic:** 11 — Governança e Qualidade  
**Status:** Done  
**Criado em:** 2026-05-06  

## Descrição

Como desenvolvedor, quero um teste de integração que verifique o caminho crítico completo do pipeline (foto → análise → copy → design) sem depender de APIs externas reais.

## Por quê

Os 191 testes existentes usam mocks e não teriam capturado nenhum dos bugs desta sessão (NameError de import, limite de 5MB, encoding Unicode). Um smoke test que ao menos importa e instancia todos os agentes fecha a lacuna mais óbvia.

## Acceptance Criteria

- [ ] Arquivo `backend/tests/test_smoke_pipeline.py` criado
- [ ] Testa import de todos os agentes sem NameError
- [ ] Testa `_compress_image_for_claude()` com imagem sintética > 4MB
- [ ] Testa `_process_card()` do designer com texto acentuado (ã, ç, é)
- [ ] Testa parsing de "Música de fundo:" no copywriter
- [ ] Todos os testes passam sem conexão com banco ou APIs externas

## Critério de cobertura

Não é cobertura de linha — é cobertura de **caminho de falha**. Cada bug desta sessão deve ter um teste correspondente que o teria detectado.
