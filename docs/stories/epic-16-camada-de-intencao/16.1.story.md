## Contexto

Para que o sistema possa recomendar formato e estratégia a partir de um objetivo de marketing, precisa existir um mapeamento declarativo que define quais combinações são mais efetivas para cada intenção.

Esta story cria esse mapeamento como arquivo de configuração TypeScript (`frontend/lib/intents.ts`) e como constante Python no backend (`copywriter.py` — contexto de intenção para o prompt).

---

## Descrição

### Frontend: `frontend/lib/intents.ts`

```typescript
export type MarketingIntent =
  | "gerar_orcamentos"
  | "ganhar_seguidores"
  | "aumentar_engajamento"
  | "construir_autoridade"
  | "manter_ativo";

export interface Intent {
  id:          MarketingIntent;
  label:       string;          // "Gerar orçamentos"
  description: string;         // "Transformar visitantes em contatos"
  icon:        string;          // emoji
  recommendation: {
    postTypeId: PostTypeId;     // formato recomendado
    strategyId: string;         // estratégia recomendada
    reason:     string;         // "Prova social é o formato de maior conversão para novos clientes"
  };
}
```

**Mapeamentos (tabela de referência para implementação):**

| Intent | Formato recomendado | Estratégia recomendada | Razão |
|--------|--------------------|-----------------------|-------|
| gerar_orcamentos | feed_photo | prova_social | Resultado entregue + CTA de orçamento é o caminho mais direto para conversão |
| ganhar_seguidores | reels | hook_choque | Reels com hook forte têm o maior alcance orgânico da plataforma |
| aumentar_engajamento | carousel | comparativo | Carrossel comparativo gera mais comentários e salvamentos |
| construir_autoridade | carousel | passo_a_passo | Tutorial técnico posiciona como especialista e gera saves |
| manter_ativo | story | bastidores_dia | Baixo esforço de produção, alta frequência possível |

### Backend: `backend/app/agents/copywriter.py`

Adicionar dicionário `INTENT_CONTEXT_PROMPTS` que enriquece o system prompt quando `marketing_intent` está presente:

```python
INTENT_CONTEXT_PROMPTS: dict[str, str] = {

```

## Riscos

- Mapeamento pode não ser o ideal para todos os nichos → mitigação: o usuário pode ignorar a recomendação e navegar manualmente (a recomendação é sugestão, não força)
- Injeção do contexto de intenção no prompt pode inflar o prompt além do ideal → mitigação: `INTENT_CONTEXT_PROMPTS` são frases curtas (1 linha cada)
