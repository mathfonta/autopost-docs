## Descrição

Como cliente AutoPost, quero poder instalar o dashboard como app no meu celular e receber
notificações quando um post precisar de aprovação — para não precisar abrir o navegador
e não perder posts aguardando.

---

- [x] AC1: App installable como PWA — manifesto válido com ícone 192px e 512px, `display: standalone`

## Dev Notes

### Web Push stack

```typescript
// lib/push.ts
export async function subscribeToPush() {
  const registration = await navigator.serviceWorker.ready;
  const subscription = await registration.pushManager.subscribe({
    userVisibleOnly: true,
    applicationServerKey: process.env.NEXT_PUBLIC_VAPID_PUBLIC_KEY,
  });
  await api.post('/push/subscribe', subscription);
}
```

Backend precisa de VAPID keys — gerar com `web-push generate-vapid-keys`.
