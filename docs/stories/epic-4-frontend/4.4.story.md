## Descrição

Como novo cliente AutoPost, preciso de um wizard guiado que me leve da criação de conta
até a conexão do Instagram — para estar pronto para receber meu primeiro post automatizado
sem precisar de suporte técnico.

## Dev Notes

### Detectar retorno do OAuth Meta

```typescript
// app/onboarding/page.tsx
const searchParams = useSearchParams();
const connected = searchParams.get('connected'); // Meta redireciona com parâmetro

useEffect(() => {
  if (connected === 'true') {
    setStep(2); // pula para etapa "concluído" ou exibe ✓
    toast.success('Instagram conectado com sucesso!');
  }
}, [connected]);
```

### Redirect para onboarding

```typescript
// middleware.ts — adicionar à lógica existente
const { data: onboardingStatus } = await fetch('/onboarding/status');
if (onboardingStatus?.status !== 'completed' && !pathname.startsWith('/onboarding')) {
  return NextResponse.redirect(new URL('/onboarding', request.url));
}
```

### Brand profile form (integração com backend)

O backend tem dois endpoints para onboarding conversacional:
- `POST /onboarding/start` — inicia sessão com `client_id`
- `POST /onboarding/message` — envia mensagem, recebe resposta do Claude

Para o wizard visual, os campos do form são enviados diretamente como mensagens estruturadas
ou salvos via `brand_profile` no Supabase (dependendo da implementação — verificar com backend).
