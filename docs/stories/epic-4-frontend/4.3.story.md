# Story 4.3 — Preview do Post (Estilo Instagram)

**Status:** Ready for Review  
**Epic:** 4 — Frontend Dashboard  
**Story:** 4.3  
**Estimativa:** 3 pontos  
**Depende de:** Story 4.2

---

## Descrição

Como cliente AutoPost, quero ver exatamente como meu post vai aparecer no Instagram
antes de aprovar — com a imagem processada, legenda, hashtags e CTA — para ter
confiança de que ficou bom antes de publicar.

---

## Critérios de Aceitação

- [ ] AC1: Rota `/posts/[id]` exibe preview completo do post em formato 1:1 (quadrado, 375px largura máxima)
- [ ] AC2: Preview imita a interface do Instagram: foto quadrada, nome do perfil com avatar placeholder, legenda com hashtags em azul
- [ ] AC3: Legenda exibe caption + CTA + hashtags separados por quebra de linha, como apareceria no Instagram
- [ ] AC4: Botões "Aprovar" e "Rejeitar" visíveis abaixo do preview (mesma lógica da Story 4.2)
- [ ] AC5: Clique no card do dashboard navega para `/posts/[id]`
- [ ] AC6: Botão "voltar" retorna ao dashboard
- [ ] AC7: Se post já publicado: exibe permalink "Ver no Instagram" e métricas disponíveis (impressões, curtidas)
- [ ] AC8: Loading skeleton enquanto carrega dados do post

## Dev Notes

### Layout do preview Instagram

```tsx
// components/InstagramPostPreview.tsx
<div className="max-w-[375px] mx-auto border rounded-lg overflow-hidden bg-white">
  {/* Header estilo Instagram */}
  <div className="flex items-center gap-2 p-3">
    <div className="w-8 h-8 rounded-full bg-gradient-to-tr from-yellow-400 to-pink-600" />
    <span className="font-semibold text-sm">{brandName}</span>
    <span className="text-gray-400 text-xs ml-auto">Agora</span>
  </div>
  
  {/* Imagem quadrada */}
  <div className="aspect-square w-full">
    <img src={imageUrl} alt="Post" className="w-full h-full object-cover" />
  </div>
  
  {/* Ações estilo Instagram */}
  <div className="flex gap-4 p-3 text-gray-800">
    <Heart className="w-6 h-6" />
    <MessageCircle className="w-6 h-6" />
    <Send className="w-6 h-6" />
  </div>
  
  {/* Legenda */}
  <div className="px-3 pb-3 text-sm">
    <span className="font-semibold mr-1">{brandName}</span>
    {caption}
    <br /><br />
    <span className="text-blue-500">{hashtags.map(h => `#${h}`).join(' ')}</span>
  </div>
</div>
```
