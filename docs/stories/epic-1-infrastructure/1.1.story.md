## Dev Notes

- `$PORT` no Dockerfile deve usar shell form: `CMD ["/bin/sh", "-c", "uvicorn app.main:app --host 0.0.0.0 --port ${PORT:-8000}"]`
- Supabase Transaction Pooler (porta 6543) requer `statement_cache_size=0` no asyncpg
- Beat usa `/tmp/celerybeat-schedule` (gravável em containers Railway)
- Variáveis de ambiente precisam ser configuradas nos 3 serviços Railway individualmente
