**Status:** In Progress

**Prioridade:** Alta

## Objetivo

Incorporar o framework "Modo Engenharia" (origem: Promove Digital) no pipeline de criação de conteúdo do AutoPost. O objetivo é fazer o copywriter e o sistema editorial pensar como o algoritmo do Instagram — gerando conteúdo que escala mesmo em perfis com zero seguidores.

## Problema

O copywriter atual gera copy tecnicamente boa (storytelling, emojis, hashtags dinâmicas via Exa) mas não considera:
- Se o hook faz sentido para quem **nunca viu o perfil**
- Se o conteúdo gera os sinais certos para o algoritmo (saves > likes)
- Qual sequência de posts maximiza alcance nos primeiros 10 conteúdos de um cliente novo
