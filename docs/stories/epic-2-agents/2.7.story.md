# Story 2.7 — OAuth Meta (Conexão Instagram/Facebook)

**Status:** Done  
**Epic:** 2 — Agentes de IA  
**Story:** 2.7  
**Estimativa:** 3 pontos  
**Depende de:** 1.2

---

## Descrição

Como cliente do AutoPost, quero conectar minha conta Instagram Business e Página do Facebook via OAuth para que o Agente Publicador possa publicar posts em meu nome de forma segura.

---

## Critérios de Aceitação

- [x] AC1: `GET /meta/connect` retorna `auth_url` para iniciar o fluxo OAuth com Meta
- [x] AC2: `GET /meta/callback` troca o `code` por Long-Lived Token e salva `instagram_business_id`, `facebook_page_id` e `meta_access_token` no Client
- [x] AC3: `GET /meta/status` retorna status da conexão (conectado/desconectado) e dados da conta
- [x] AC4: Token de curta duração é trocado por Long-Lived Token (~60 dias de validade)
- [x] AC5: State token com JWT para prevenir CSRF no callback
- [x] AC6: Testes cobrindo callback, troca de token e tratamento de erros

---

## Tasks
