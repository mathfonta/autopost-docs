## Descrição

Como AutoPost, preciso que os agentes aprendam com cada post publicado — escrevendo resultados no cérebro local após coletar métricas, lendo padrões acumulados antes de agir, e gerando insights semanais automáticos com Claude — para que cada post novo seja melhor que o anterior.

---

- [x] AC1: Após `collect_metrics`, resultados são escritos em `.cerebro-autopost/historico/{YYYY-WW}.md` com: tipo de foto, legenda resumida, horário publicado, curtidas, alcance, comentários
- [x] AC2: `analyze_photo_with_ai` lê `.cerebro-autopost/PADROES.md` antes de gerar análise — se arquivo existe e tem conteúdo, inclui como contexto adicional no prompt do Claude

- [x] AC5: `PADROES.md` gerado pelo Claude contém: tipo de foto de melhor desempenho, melhor horário, CTA mais eficaz, recomendações para próxima semana
