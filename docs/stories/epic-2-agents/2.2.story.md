## Critérios de Aceitação

- [x] AC1: Task `generate_copy` chama Claude Sonnet com `analysis_result` + `brand_profile` e retorna copy estruturado
- [x] AC2: `copy_result` contém: `caption` (≤2200 chars), `hashtags` (lista), `cta`, `suggested_time` (horário sugerido HH:MM)

- [x] AC6: Claude Sonnet — modelo `claude-sonnet-4-6` para qualidade máxima de copy
- [x] AC7: Testes com mock da API Anthropic
