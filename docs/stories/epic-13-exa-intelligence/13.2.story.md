## User Story

**As a** usuário do AutoPost que publica sobre construção civil,  
**I want** que a copy gerada mencione tendências reais do meu setor,  
**so that** meus posts sejam relevantes para o momento atual do mercado sem eu precisar pesquisar nada.

---

## Descrição

Conecta o Exa ao pipeline: executa `search_exa_trends()` em paralelo ao `analyze_photo/video`
e injeta o resultado como `#TENDENCIAS_DO_NICHO` no prompt do `copywriter.py`.

O usuário não vê essa etapa. A copy simplesmente chega mais contextualizada com o mercado real.

---

## Acceptance Criteria

- [ ] AC1: `generate_copy_with_ai()` em `copywriter.py` recebe novo parâmetro `exa_context: str | None = None`
  - Se não None: injeta `\n\n#TENDENCIAS_DO_NICHO (dados Exa, últimos 30 dias):\n{exa_context}` no prompt, após `#CONTEXTO_DO_USUARIO`
  - Se None: copy gerada normalmente — **zero breaking change**
- [ ] AC2: Nova coluna `exa_trends_context TEXT NULL` em `content_requests` — migration backward compatible
- [ ] AC3: Em `pipeline.py`, a busca Exa é disparada **em paralelo** ao analyze:
  ```python
  analyze_result, exa_context = await asyncio.gather(
      analyze_photo_with_ai(...),
      search_exa_trends(segment=client.segment, content_type=...)
  )
  ```
  Ou, se já for Celery task chain, usar `group` do Celery para paralelismo.
- [ ] AC4: `exa_trends_context` salvo no banco após generate_copy (para auditoria e eventual exibição)
- [ ] AC5: Log `[exa] tendências encontradas: {N} chars para segment={segment}` ou `[exa] sem tendências — continuando normalmente`
- [ ] AC6: Se `EXA_PROVIDER=disabled`: `search_exa_trends()` retorna `None` imediatamente, pipeline sem overhead

## Prompt diff (antes/depois)

```python
# ANTES — copywriter.py (sem Exa)
prompt = f"""
Gere copy para Instagram.
#CONTEXTO_DO_USUARIO: {user_context}
#TOM_DE_VOZ: {voice_tone}
#TRANSCRIÇÃO: {audio_transcript}
"""

# DEPOIS — com Exa injetado
exa_block = f"\n\n#TENDENCIAS_DO_NICHO (dados Exa, últimos 30 dias):\n{exa_context}" if exa_context else ""
prompt = f"""
Gere copy para Instagram.
#CONTEXTO_DO_USUARIO: {user_context}
#TOM_DE_VOZ: {voice_tone}
#TRANSCRIÇÃO: {audio_transcript}{exa_block}

INSTRUÇÕES:
- Se houver #TENDENCIAS_DO_NICHO, use UMA referência sutil a tendências reais
- Não cite fontes nem URLs no post
- Mantenha foco na obra/serviço da empresa
"""
```

## Exemplo de contexto Exa injetado

```
#TENDENCIAS_DO_NICHO (dados Exa, últimos 30 dias):
• Porcelanato grande formato (90x90cm) domina menções em obras residenciais de alto padrão em 2025
• Vídeos "antes e depois" geram 4x mais saves que fotos estáticas em reforma civil
• Tom pessoal e próximo supera técnico em 67% dos posts virais de construção
```
