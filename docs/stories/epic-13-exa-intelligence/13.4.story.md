## User Story

**As a** usuário do AutoPost,  
**I want** ver um resumo semanal do que está em alta no meu segmento de mercado,  
**so that** eu possa planejar meus posts da semana com base em tendências reais sem precisar pesquisar em outras ferramentas.

---

## Descrição

Task Celery agendada (Celery Beat — toda segunda às 07:00 horário de Brasília) que busca
tendências da semana no Exa para todos os segmentos de clientes ativos.

O resultado é salvo em `weekly_context` no Supabase e exibido como card
"Inteligência de mercado" no dashboard — **única story do Epic 13 com output visível ao usuário**.

A busca semanal também alimenta a Story 13.3 com hashtags frescas sem gasto de API por post.

## Implementação da task

```python
# app/tasks/pipeline.py (adicionar à beat_schedule)
from celery.schedules import crontab

# Em app/celery.py ou onde está o beat_schedule:
CELERY_BEAT_SCHEDULE = {
    # ... entradas existentes ...
    "weekly-content-intelligence": {
        "task": "pipeline.generate_weekly_intelligence",
        "schedule": crontab(hour=7, minute=0, day_of_week="monday"),
        "options": {"timezone": "America/Sao_Paulo"}
    }
}

# A task em si:
@celery_app.task(name="pipeline.generate_weekly_intelligence")
async def generate_weekly_intelligence():
    if settings.EXA_PROVIDER != "exa":
        logger.info("[weekly-intel] EXA_PROVIDER=disabled — pulando")
        return

    queries = [
        "construção civil Brasil notícias semana",
        "tendências acabamento revestimento 2025",
        "mercado imobiliário Florianópolis Santa Catarina",
    ]

    all_snippets = []
    for q in queries:
        result = await search_exa_raw(q, days_back=7)  # retorna lista de snippets
        all_snippets.extend(result)

    summary = await _summarize_with_gemini(all_snippets)
    hashtags = _extract_hashtags(all_snippets)

    await save_weekly_context(
        week_of=date.today(),
        segment="Construção civil",  # v1: fixo, v2: por segmento de cliente
        raw_snippets=all_snippets,
        summary=summary,
        hashtags=hashtags
    )
    logger.info(f"[weekly-intel] concluído: {len(all_snippets)} snippets, summary={len(summary)} chars")
```
