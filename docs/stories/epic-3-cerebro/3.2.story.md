## Dev Notes

### Prompt de classificação
```
Você recebe padrões descobertos no projeto {project_name} (segmento: {segment}).
Para cada padrão, classifique:
- universal: funciona independente de segmento, plataforma ou audiência
- nicho: funciona especificamente para o segmento {segment}
- local: específico deste cliente, não generalizável

Retorne JSON: [{"pattern": "...", "type": "universal|nicho|local", "confidence": 0.0-1.0}]
```

### Path resolution
```python
GLOBAL_CEREBRO = Path(r"C:\Users\Matheus\OneDrive\Obsidian\🌐 Global")
LOCAL_CEREBRO = Path(os.getenv("CEREBRO_PATH", r"C:\Users\Matheus\OneDrive\Obsidian\AutoPost\🧠 Cerebro"))
```

### Merge strategy
- Lê PADROES_GLOBAIS.md atual
- Para cada padrão novo: verifica similaridade semântica (via Claude) com existentes
- Se similar (>80%): atualiza frequência/confiança do existente
- Se novo: adiciona na seção correta com `[Fonte: {project_name}, {data}]`
