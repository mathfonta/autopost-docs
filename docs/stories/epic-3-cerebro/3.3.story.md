### Regra cerebro-global.md
```markdown
# Cérebro Global — Regras de Acesso

Localização: C:\Users\Matheus\OneDrive\Obsidian\🌐 Global\

## Quando ler
- @architect: lê APRENDIZADOS.md (seção Técnico) ao iniciar qualquer design
- @pm: lê PADROES_GLOBAIS.md e projetos/ ao criar PRD ou epic
- @analyst: lê PADROES_GLOBAIS.md e seção do nicho relevante
- @dev: lê APRENDIZADOS.md (seção Técnico) quando enfrentar problema não trivial

## Quando escrever
- Qualquer agente pode usar *cerebro-update para registrar descobertas importantes
- Discoveries são adicionadas com: data, projeto, agente, contexto

## Formato de leitura
Ao ler o cérebro global, priorize:
1. Padrões com alta confiança (>0.8)
2. Aprendizados que se aplicam ao contexto atual
3. Lições de projetos no mesmo segmento/domínio
```

`*cerebro` e `*cerebro-update` **não são skills AIOS registradas** — são instruções comportamentais adicionadas ao arquivo `~/.claude/rules/cerebro-global.md`. O @dev deve expandir esse arquivo com:
- Seção "Comando *cerebro": instrui o agente a ler os arquivos do cérebro global e formatar o resumo abaixo
- Seção "Comando *cerebro-update": instrui o agente a appender em APRENDIZADOS.md com data/projeto/agente
