## Descrição

A Story 9.4-análise implementou extração de frames para que o Claude Haiku "veja" o vídeo visualmente. Mas quando o vídeo tem fala (profissional explicando um serviço, dica de obra, depoimento), a IA não ouve o conteúdo — apenas olha para quadros estáticos.

Esta story adiciona transcrição de áudio usando OpenAI Whisper API. Quando um vídeo (reels/story) é enviado, o sistema:
1. Extrai o áudio via FFmpeg (já instalado no Docker)
2. Envia ao Whisper API (`whisper-1`)
3. Passa a transcrição ao Agente Analista como contexto adicional
4. O Copywriter usa a transcrição real do vídeo para criar uma legenda precisa

Fallback gracioso: se a transcrição falhar (sem `OPENAI_API_KEY`, vídeo mudo, timeout), o pipeline continua normalmente com apenas análise visual.

## Dev Notes

### Extração de áudio

FFmpeg já está no Dockerfile. Extração do áudio em formato compatível com Whisper (mp3, max 25MB):

```python
def _transcribe_video_audio_sync(video_bytes: bytes) -> str | None:
    import tempfile, subprocess, os
    from openai import OpenAI
    
    try:
        client = OpenAI()  # usa OPENAI_API_KEY do env
    except Exception:
        return None  # sem chave configurada
    
    with tempfile.TemporaryDirectory() as tmpdir:
        video_path = os.path.join(tmpdir, "input.mp4")
        audio_path = os.path.join(tmpdir, "audio.mp3")
        
        with open(video_path, "wb") as f:
            f.write(video_bytes)
        
        # Extrai áudio: mono, 16kHz, limitado a 24MB
        result = subprocess.run(
            ["ffmpeg", "-i", video_path, "-vn", "-ar", "16000",
             "-ac", "1", "-b:a", "32k", "-fs", "24000000", audio_path],
            capture_output=True, timeout=30,
        )
        
        if result.returncode != 0 or not os.path.exists(audio_path):
            return None
        
        audio_size = os.path.getsize(audio_path)
        if audio_size < 1000:  # menos de 1KB = provavelmente mudo
            return None
        
        with open(audio_path, "rb") as f:
            transcript = client.audio.transcriptions.create(
                model="whisper-1",
                file=f,
                language="pt",
            )
        
        return transcript.text.strip() or None
    
    return None
```

## Riscos

| Risco | Probabilidade | Impacto | Mitigação |
|-------|--------------|---------|-----------|
| OPENAI_API_KEY não configurada | Alta (inicialmente) | Baixo | Fallback gracioso — funciona sem a chave |
| Vídeo mudo → Whisper retorna string vazia | Média | Baixo | Verificar `audio_size < 1000` antes de chamar |
| Custo inesperado Whisper | Baixa | Médio | $0.006/min — neglicível em escala MVP |
| Timeout em vídeos longos | Baixa | Médio | `timeout=30` no subprocess de extração + try/except |
