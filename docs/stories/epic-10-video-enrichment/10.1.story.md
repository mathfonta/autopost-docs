**Epic:** 10 — Enriquecimento de Conteúdo de Vídeo

## Descrição

Clientes querem especificar a música de fundo ao preparar um post — principalmente Reels e Stories falados. Hoje não há campo para isso: o usuário preenche apenas o "Descreva o conteúdo", sem mencionar a trilha sonora.

Esta story adiciona um campo opcional "Música de fundo" no ContextModal (frontend). O valor é mesclado ao `user_context` antes do envio — sem qualquer mudança de schema no banco —, e o Copywriter já recebe essa informação e pode incorporar sugestão de música na legenda.

## Dev Notes

O campo de música é mesclado no frontend antes de chamar `onConfirm(context)`. O assinatura atual do callback é `onConfirm: (context: string) => void` — permanece inalterada. A concatenação acontece dentro do `handleConfirm()`:

```typescript
function handleConfirm() {
  let full = context.trim();
  if (music.trim()) {
    full = [full, `Música de fundo: ${music.trim()}`].filter(Boolean).join("\n\n");
  }
  setContext("");
  setMusic("");
  onConfirm(full);
}
```

O Copywriter recebe `user_context` com a informação de música e pode mencionar na legenda (ex: "🎵 Neon Blade - Gustavo Mioto").
